# Zend3bcHelper
A Zend Framework 3 backwards compatibility helper modul. Get many behavior from ZF 2.5 on ZF 3

using composer:
composer.json
<pre>{
    ...

    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/MNHcC/Zend3bcHelper.git"
        }
    ],
    "require": {
        "php": ">=5.6.0",
        "MNHcC/Zend3bcHelper": "dev-master"
    }

}</pre>
